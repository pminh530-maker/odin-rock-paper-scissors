# odin-rock-paper-scissors
In this project, I learned how to apply some different features of JavasScript, such as:
a. Creating global variables and assign values to them
b. Using statements (if ... else and return)
c. Creating and calling functions
d. Using a loop
e. Incrementing a value
f. Debugging (by reading the console message, I did not really use the debugger at all. I was not comfortable with using all the features like adding breakpoints and stepping in & out of functions yet)

The vast majority of the project was created on my own, but I needed some AI assistance to implement the loop & displaying a round counter in the end.
